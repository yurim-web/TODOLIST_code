# 🌟 Personal TodoList & Weather Dashboard

개인용 할 일 관리, 날씨 정보, 일기 작성 기능을 제공하는 완전 반응형 웹 애플리케이션입니다.

## 🌐 배포 사이트

**🔗 [https://yurim-web.github.io/todolist/](https://yurim-web.github.io/todolist/)**

> 🎉 **새롭게 리뉴얼된 버전입니다!** (배포일: 2025.09.23)

## ✨ 주요 기능

### 🎯 TodoList 관리
- **할 일 추가**: 간편한 입력으로 새로운 할 일 추가
- **할 일 삭제**: 클릭 한 번으로 완료된 할 일 제거
- **실시간 업데이트**: 즉시 반영되는 할 일 목록
- **반응형 입력**: 모든 화면 크기에서 최적화된 입력 경험

### 🌤️ 실시간 날씨 정보
- **현재 날씨**: 서울 지역 기준 실시간 날씨 정보
- **체감온도**: 실제 체감 온도 표시
- **날씨 상태**: 현재 날씨 상태 설명
- **시각적 아이콘**: 직관적인 날씨 아이콘
- **반응형 레이아웃**: 화면 크기에 따른 최적화된 표시

### 👤 사용자 인증
- **별명 설정**: 개인화된 사용자 경험
- **예쁜 모달**: 현대적인 UI/UX의 모달창 (애니메이션 포함)
- **로그아웃 기능**: 새로운 별명으로 재로그인
- **반응형 모달**: 모바일에서도 완벽한 모달 경험

### 📝 일기 작성 기능
- **한 줄 일기**: 간편한 일기 작성 및 저장
- **실시간 표시**: 입력한 일기가 즉시 화면에 표시
- **반응형 입력**: 모바일/태블릿/데스크톱 최적화
- **직관적 UI**: Todo와 동일한 스타일의 일관된 디자인

### 📅 추가 기능
- **실시간 시계**: 현재 날짜와 시간 표시
- **로그 기록**: 활동 기록 관리
- **완전 반응형**: 모든 디바이스에서 완벽한 경험

## 🎨 디자인 특징

### 📱 완전 반응형 디자인
- **모바일 우선**: 모바일에서 최적화된 경험 (767px 이하)
- **태블릿 지원**: 중간 크기 화면에 최적화 (768px - 1023px)
- **데스크톱 호환**: 큰 화면에서도 완벽한 레이아웃 (1024px 이상)
- **유연한 레이아웃**: 화면 크기에 따른 자동 레이아웃 변경

### 🎭 아름다운 UI/UX
- **겨울 테마**: 아름다운 겨울 배경 이미지
- **글래스모피즘**: 반투명 효과로 현대적인 느낌
- **부드러운 애니메이션**: 자연스러운 전환 효과 및 호버 효과
- **직관적 인터페이스**: 사용하기 쉬운 디자인
- **일관된 스타일**: 모든 컴포넌트의 통일된 디자인 언어

## 🛠️ 기술 스택

- **Frontend**: React 18
- **UI Library**: Radix UI
- **Styling**: CSS3 (반응형 디자인)
- **API**: OpenWeatherMap API
- **Build Tool**: Create React App

## 🚀 시작하기

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 빌드

```bash
# 프로덕션 빌드
npm run build
```

## 📁 프로젝트 구조

```
src/
├── components/
│   ├── common/
│   │   ├── Container.jsx      # 공통 컨테이너 컴포넌트
│   │   ├── Datetime.jsx       # 날짜/시간 표시
│   │   ├── Diary.jsx          # 일기 작성
│   │   ├── Log.jsx            # 사용자 로그인/로그아웃
│   │   ├── NameModal.jsx      # 별명 입력 모달
│   │   ├── Weather.jsx        # 날씨 정보
│   │   └── Todolist/
│   │       ├── Todo.jsx       # 메인 Todo 컴포넌트
│   │       ├── Todoinput.jsx  # Todo 입력/목록
│   │       └── Todotitle.jsx  # Todo 제목
│   └── layout/
│       ├── Left.jsx           # 왼쪽 레이아웃
│       └── Right.jsx          # 오른쪽 레이아웃
├── css/
│   ├── reset.css              # CSS 리셋
│   └── responsive.css         # 반응형 스타일
└── App.jsx                    # 메인 앱 컴포넌트
```

## 🎯 주요 컴포넌트 설명

### Log.jsx
- 사용자 인증 및 별명 관리
- 예쁜 모달창으로 사용자 경험 향상
- 로그인/로그아웃 기능
- 반응형 space-between 레이아웃

### Weather.jsx
- OpenWeatherMap API 연동
- 실시간 날씨 정보 표시
- 서울 지역 기준 날씨 데이터
- 반응형 이미지 및 정보 배치

### Diary.jsx
- 한 줄 일기 작성 및 저장 기능
- 실시간 텍스트 표시
- Todo와 동일한 반응형 입력 스타일
- 직관적인 사용자 인터페이스

### Todoinput.jsx
- 할 일 추가/삭제 기능
- 실시간 상태 관리
- 직관적인 사용자 인터페이스
- 완전 반응형 입력 시스템

### NameModal.jsx
- 재사용 가능한 모달 컴포넌트
- 부드러운 애니메이션 효과
- 완전 반응형 디자인
- 키보드 지원 (Enter 키)

`src/css/responsive.css`에서 색상, 레이아웃, 애니메이션을 커스터마이징할 수 있습니다.

- 반응형별로 명확하게 구분된 CSS 구조

### 배경 이미지 변경

### 컴포넌트 수정
- `src/components/common/` 폴더에서 각 기능별 컴포넌트 수정 가능
- `src/components/layout/` 폴더에서 레이아웃 구조 변경 가능
